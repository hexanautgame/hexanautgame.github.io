Privacy Policy for Hexanaut Game

This privacy policy describes how information is collected and used when you visit and interact with the Hexanaut Game website (https://hexanautgame.github.io/index.html).

Information Collection and Use

This website is a static HTML page hosted on GitHub Pages. As such, it does not directly collect or store any personally identifiable information (PII) from its visitors. There are no user accounts, login systems, or forms that request personal data.

However, please be aware of the following:

  • GitHub Pages: The website is hosted on GitHub Pages. GitHub may collect certain technical information, such as IP addresses and browser details, for operational and security purposes. Please refer to GitHub's privacy policy for more information on their data collection practices.
  • Third-Party Services: If the game embeds or links to any third-party services (e.g., analytics, advertising, or game engines), those services may have their own privacy policies. We encourage you to review the privacy policies of any third-party services you interact with.
  • Cookies: This static webpage itself does not set cookies. However, GitHub pages and any embedded third party services might. Please check the privacy policies of those services.

Data Security

As this website does not collect or store personal data, there are no specific security measures in place to protect such data. However, GitHub Pages provides basic security measures to protect the integrity of the website's content.

Children's Privacy

This website is not specifically directed at children under the age of 13. We do not knowingly collect personal information from children. If you believe that we have inadvertently collected information from a child, please contact us so that we can take appropriate action.

Changes to This Privacy Policy

We may update this privacy policy from time to time. We will post any changes on this page. Your continued use of the website after any changes constitutes your acceptance of the new privacy policy.

Contact Us

If you have any questions about this privacy policy, please contact us by submitting an issue on the GitHub repository where this site is hosted. Because this is a static github page, there is no direct contact method.
